# Исследование надежности заемщиков

## Описание проекта
Анализ факторов, влияющих на возврат кредита в срок.  
Исследование проведено на основе данных о клиентах банка.

## Данные
- Датасет содержит информацию о 21 525 клиентах
- Поля: количество детей, трудовой стаж, возраст, образование, семейное положение, пол, тип дохода, наличие задолженности, доход, цель кредита

## Задачи
1. Предобработка данных: обработка пропусков, аномалий, дубликатов
2. Категоризация: доход, цель кредита, количество детей
3. Анализ влияния различных факторов на возврат кредита

## Инструменты
- Python (Pandas, NumPy)
- Jupyter Notebook

## Результаты
Выявлены группы клиентов с наибольшим риском невозврата кредита:

| Фактор | Группа риска | Доля должников |
|--------|-------------|----------------|
| Количество детей | 1-2 детей | 10.3% |
| Семейное положение | Не женат / не замужем | 9.8% |
| Уровень дохода | Низкий (категория E) | 9.1% |
| Цель кредита | Автомобиль / Образование | 9.3% |
