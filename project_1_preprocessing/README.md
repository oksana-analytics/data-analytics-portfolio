## 📌 Описание проекта

Анализ факторов, влияющих на возврат кредита в срок.  
Исследование проведено на основе данных о клиентах банка с целью выявления групп риска для оптимизации кредитной политики.

### 🎯 Цель исследования
Определить, какие характеристики заемщиков (количество детей, семейное положение, уровень дохода, цель кредита) связаны с повышенным риском невозврата кредита.

---

## 📊 Данные

Датасет содержит информацию о **21 525** клиентах банка.

| Поле | Описание |
|------|----------|
| `children` | Количество детей |
| `days_employed` | Трудовой стаж (дней) |
| `dob_years` | Возраст (лет) |
| `education` | Уровень образования |
| `education_id` | Идентификатор образования |
| `family_status` | Семейное положение |
| `family_status_id` | Идентификатор семейного положения |
| `gender` | Пол |
| `income_type` | Тип занятости |
| `debt` | Наличие задолженности (0/1) |
| `total_income` | Ежемесячный доход |
| `purpose` | Цель кредита |

---

## 🔍 Задачи исследования

1. **Предобработка данных**
   - Обработка пропусков (медианное заполнение по типу занятости)
   - Удаление аномалий (отрицательный стаж, нереалистичное количество детей)
   - Приведение типов данных
   - Удаление дубликатов

2. **Категоризация данных**
   - Разбивка дохода на категории (A–E)
   - Группировка целей кредита (автомобиль, недвижимость, свадьба, образование)
   - Классификация по количеству детей

3. **Анализ влияния факторов**
   - Количество детей и возврат кредита
   - Семейное положение и возврат кредита
   - Уровень дохода и возврат кредита
   - Цель кредита и возврат кредита

## 📈 Ключевые выводы

### 1. Влияние количества детей
| Категория | Доля должников |
|-----------|----------------|
| До 2-х детей | **10.3%** |
| 3 и более детей | 8.9% |
| Детей нет | 8.2% |

➡️ **Вывод**: Наибольший риск невозврата у заемщиков с 1–2 детьми

### 2. Влияние семейного положения
| Семейное положение | Доля должников |
|-------------------|----------------|
| Не женат / не замужем | **9.8%** |
| Гражданский брак | 9.3% |
| Женат / замужем | 7.6% |
| В разводе | 7.1% |
| Вдовец / вдова | 6.6% |

➡️ **Вывод**: Одинокие заемщики имеют более высокий процент задолженности

### 3. Влияние уровня дохода
| Категория дохода | Диапазон | Доля должников |
|-----------------|----------|----------------|
| E | 0–30 000 руб. | **9.1%** |
| C | 50 001–200 000 руб. | 8.5% |
| A | > 1 000 000 руб. | 8.0% |
| B | 200 001–1 000 000 руб. | 7.1% |
| D | 30 001–50 000 руб. | 6.0% |

➡️ **Вывод**: Низкий доход коррелирует с повышенным риском

### 4. Влияние цели кредита
| Цель кредита | Доля должников |
|-------------|----------------|
| Операции с автомобилем | **9.3%** |
| Получение образования | **9.3%** |
| Проведение свадьбы | 7.9% |
| Операции с недвижимостью | 7.3% |

➡️ **Вывод**: Кредиты на автомобиль и образование требуют более тщательной проверки

## 💡 Рекомендации для банка

1. **Ужесточить проверку** для заемщиков с 1–2 детьми и одиноких клиентов
2. **Повысить требования** к заемщикам с низким доходом (категория E)
3. **Усилить скоринг** для кредитов на автомобиль и образование
4. **Снизить требования** к заемщикам с высоким доходом и кредитами на недвижимость
