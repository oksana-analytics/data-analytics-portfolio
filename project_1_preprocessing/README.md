## 🎯 О чем этот проект?

Банк выдает кредиты, но не все клиенты возвращают их вовремя.  
Мы проанализировали данные **21 525 заемщиков**, чтобы понять:

- **Какие клиенты чаще всего не платят?**
- **На какие характеристики обращать внимание при выдаче кредита?**
- **Какие цели кредита связаны с высоким риском?**

Результаты помогут банку снизить количество просрочек и скорректировать кредитную политику.

## 📊 Данные

У нас есть информация о клиентах:

| Поле | Что значит |
|------|------------|
| `children` | Количество детей |
| `days_employed` | Трудовой стаж (дней) |

| `family_status` | Семейное положение |

| `total_income` | Ежемесячный доход |
| `purpose` | На что взял кредит (жилье, машина, образование и т.д.) |

## 🔍 Как устроено исследование?

### Часть 1. Предобработка данных
*   **Обработка пропусков:** доход заполняли медианой по типу занятости (сотрудник, пенсионер и т.д.)
*   **Удаление аномалий:** отрицательный трудовой стаж → взяли по модулю; убрали строки с -1 и 20 детьми
*   **Приведение типов:** доход перевели из float в int
*   **Удаление дубликатов:** убрали 71 дублирующуюся строку

### Часть 2. Категоризация данных
*   **Доход** разбили на категории от A (самый высокий) до E (самый низкий)
*   **Цели кредита** объединили в 4 группы: автомобиль, недвижимость, свадьба, образование
*   **Количество детей** разбили на: нет детей, до 2-х детей, 3 и более детей

### Часть 3. Анализ факторов риска
*   **Сравнение доли должников в разных группах**
*   **Выявление самых рискованных категорий**

## 📈 Ключевые выводы

### 1. Количество детей
| Категория | Доля должников |
|-----------|----------------|
| **1–2 детей** | **10.3%** |
| 3 и более детей | 8.9% |
| Нет детей | 8.2% |

➡️ **Самый высокий риск — у заемщиков с 1–2 детьми**

### 2. Семейное положение
| Семейное положение | Доля должников |
|-------------------|----------------|
| **Не женат / не замужем** | **9.8%** |
| Гражданский брак | 9.3% |
| Женат / замужем | 7.6% |
| В разводе | 7.1% |
| Вдовец / вдова | 6.6% |

➡️ **Одинокие заемщики чаще не возвращают кредиты**

### 3. Уровень дохода
| Категория | Доход в месяц | Доля должников |
|-----------|---------------|----------------|
| **E** | 0–30 000 руб. | **9.1%** |
| C | 50 001–200 000 руб. | 8.5% |
| A | > 1 000 000 руб. | 8.0% |
| B | 200 001–1 000 000 руб. | 7.1% |
| D | 30 001–50 000 руб. | 6.0% |

➡️ **Чем ниже доход — тем выше риск**

### 4. Цель кредита
| Цель | Доля должников |
|------|----------------|
| **Операции с автомобилем** | **9.3%** |
| **Получение образования** | **9.3%** |
| Проведение свадьбы | 7.9% |
| Операции с недвижимостью | 7.3% |

➡️ **Кредиты на машину и образование — самые рискованные**

---

## 💡 Что это значит для банка

| Фактор | Рекомендация |
|--------|--------------|
| **1–2 детей** | Проверять таких клиентов строже |
| **Одинокие заемщики** | Повышенное внимание при одобрении |
| **Низкий доход (категория E)** | Возможно, стоит пересмотреть лимиты |
| **Автомобиль / образование** | Более тщательный скоринг по этим целям |
| **Недвижимость / свадьба** | Менее рискованные цели — можно лояльнее |
