## 🎯 О чём этот проект?

Инвесторы хотят открыть новое заведение общественного питания в Москве, но пока не определились с концепцией, местоположением и ценовой политикой.

**Задача исследования** — проанализировать рынок общепита Москвы, выявить ключевые особенности и дать рекомендации для открытия кофейни.

📊 **Презентация с выводами:** [Смотреть презентацию](https://disk.yandex.ru/i/5kxF7vO7KZ6iIg)

---

## 📁 Откуда данные?

Датасет с информацией о заведениях общественного питания Москвы, собранный из сервисов **Яндекс Карты** и **Яндекс Бизнес** (лето 2022 года).

**Всего в данных:** 8 406 заведений

| Поле | Что означает |
|------|--------------|
| `name` | Название заведения |
| `address` | Адрес |
| `category` | Категория (кафе, ресторан, кофейня, бар и т.д.) |
| `hours` | Часы работы |
| `lat`, `lng` | Координаты (широта, долгота) |
| `rating` | Рейтинг заведения |
| `price` | Категория цен |
| `avg_bill` | Средний чек (в виде строки) |
| `middle_avg_bill` | Средний чек (число) |
| `chain` | Сетевое заведение (1) или нет (0) |
| `district` | Административный округ |
| `seats` | Количество посадочных мест |

---

## 🔧 Подготовка данных

### Что сделали:
- Привели текст в столбцах к нижнему регистру
- Удалили 4 неявных дубликата (совпадения по названию и адресу)
- Создали столбец `street` — название улицы из адреса
- Создали столбец `is_24_7` — работает ли заведение круглосуточно

### Пропуски:
- Пропуски в `hours` и `seats` оставили как есть — это индивидуальные данные
- Пропуски в `price` не заполняли — значения различаются по округам
- Пропуски в `avg_bill` распределены по столбцам `middle_avg_bill` и `middle_coffee_cup`

Типы данных корректны.

## 💡 Рекомендации для инвесторов

### Рекомендуемый округ для открытия кофейни:
**Юго-Западный административный округ**

**Почему:**
1. **Самая высокая стоимость чашки капучино** (198 ₽) — можно установить конкурентную цену
2. **Невысокая концентрация кофеен** — меньше конкуренции, чем в центре
3. **Средний чек в округе** — 600 ₽ (не максимальный и не минимальный, золотая середина)
