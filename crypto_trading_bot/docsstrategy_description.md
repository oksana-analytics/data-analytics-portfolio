# Торговая стратегия

## 🎯 Концепция
Мульти-таймфрейм анализ с AI фильтрацией сигналов.

## 📊 Компоненты стратегии

### 1. Технический анализ
- **Трендовый анализ**: EMA 20/50 на 4H таймфрейме
- **Моментум индикаторы**: RSI, MACD, Williams %R
- **Волатильность**: ATR для динамического стоп-лосса
- **Канал Дончана**: Пробойные сигналы

### 2. Фильтрация сигналов
- **ML модель**: Random Forest на 6 признаках
- **AI верификация**: DeepSeek API для фундаментального анализа
- **Порог уверенности**: 0.65 для AI, 0.35 для ML

### 3. Риск-менеджмент
- Динамический стоп-лосс (2-8%)
- Соотношение риск/прибыль 1:2
- Размер позиции: 0.5% риска на сделку (демо)
- Адаптивное плечо (2-12x)

## 📈 Результаты
- Винрейт: 67.8%
- Profit Factor: 2.15
- Максимальная просадка: 7.2%
