## 📌 Описание проекта

Анализ бизнес-показателей развлекательного приложения **Procrastinate Pro+**. Несмотря на большие вложения в рекламу, компания последние несколько месяцев терпит убытки. Исследование направлено на выявление причин неэффективности рекламных кампаний и поиск точек роста.

### 🎯 Цель исследования
Разобраться в причинах убытков и предложить рекомендации для выхода компании в плюс за счёт оптимизации маркетинговой стратегии.

---

## 📊 Данные

Проект использует три набора данных:

| Файл | Описание |
|------|----------|
| `visits_info_short.csv` | Лог сервера с информацией о посещениях (309 901 запись) |
| `orders_info_short.csv` | Информация о заказах (40 212 записей) |
| `costs_info_short.csv` | Информация о расходах на рекламу (1 800 записей) |

**Период данных:** 1 мая – 27 октября 2019 года

---

## 🔍 Задачи исследования

1. **Предобработка данных**
   - Приведение названий столбцов к единому формату (snake_case)
   - Преобразование типов данных (даты → datetime)
   - Проверка на пропуски и дубликаты

2. **Создание функций для расчёта метрик**
   - `get_profiles()` — формирование профилей пользователей
   - `get_retention()` — расчёт удержания (Retention Rate)
   - `get_conversion()` — расчёт конверсии в покупателей
   - `get_ltv()` — расчёт LTV и ROI
   - Функции для визуализации метрик

3. **Исследовательский анализ данных**
   - Анализ распределения пользователей по странам, устройствам и рекламным каналам
   - Выявление ключевых сегментов

4. **Маркетинговый анализ**
   - Расчёт общих расходов на маркетинг
   - Анализ динамики расходов по рекламным каналам
   - Расчёт стоимости привлечения одного пользователя (CAC) по каналам

5. **Оценка окупаемости рекламы**
   - Анализ LTV, ROI и CAC в целом
   - Анализ конверсии и удержания пользователей
   - Анализ окупаемости с разбивкой по странам, каналам и устройствам

## 🚨 Причины неэффективности привлечения пользователей

1. **Слишком высокая стоимость привлечения (CAC)**
   - Основные каналы (TipTop, FaceBoom) имеют самый высокий CAC
   - Пользователи из United States обходятся дороже, чем из других стран

2. **Неоптимальное распределение бюджета**
   - 81% бюджета уходит на два канала, которые не окупаются
   - Недостаточное финансирование более эффективных каналов

3. **Отсутствие данных о событиях (events)**
   - Нет возможности проверить технические ошибки и поведение пользователей внутри приложения

## 💡 Рекомендации для отдела маркетинга

### 1. Оптимизация бюджета
- **Сократить расходы** на каналы **TipTop** и **FaceBoom** — они не окупаются
- **Перераспределить бюджет** на более эффективные каналы:
  - `lambdaMediaAds`
  - `WahooNetBanner`
  - `RocketSuperAds`
  - `OppleCreativeMedia`
  - `YRabbit`
  - `MediaTornado`
  - `LeapBob`

### 2. Географическая диверсификация
- **Увеличить привлечение** пользователей из **UK, France и Germany**
- Эти страны показывают лучшую окупаемость по сравнению с United States

### 3. Диверсификация по устройствам
- **Усилить привлечение** пользователей с устройствами **PC и Android**
- Эти сегменты демонстрируют более эффективную окупаемость

### 4. Технические улучшения
- **Добавить сбор данных о событиях (events)** для более глубокого анализа
- Это позволит выявлять технические ошибки и оптимизировать пользовательский путь
