## 📌 Описание проекта

Анализ исторических данных о продажах компьютерных игр. Исследование закономерностей, определяющих успешность игры, для планирования рекламных кампаний на 2017 год.

### 🎯 Цель исследования
Выявить ключевые факторы успеха видеоигр: платформы, жанры, влияние оценок критиков и пользователей, региональные особенности. На основе анализа дать рекомендации для маркетинговой стратегии на 2017 год.

---

## 📊 Данные

**Источник данных:**  
[games.csv](https://code.s3.yandex.net/datasets/games.csv) — открытый датасет с информацией о продажах компьютерных игр

| Поле | Описание |
|------|----------|
| `name` | Название игры |
| `platform` | Платформа (Xbox, PlayStation, PC и др.) |
| `year_of_release` | Год выпуска |
| `genre` | Жанр игры |
| `na_sales` | Продажи в Северной Америке (млн копий) |
| `eu_sales` | Продажи в Европе (млн копий) |
| `jp_sales` | Продажи в Японии (млн копий) |
| `other_sales` | Продажи в других странах (млн копий) |
| `critic_score` | Оценка критиков (макс. 100) |
| `user_score` | Оценка пользователей (макс. 10) |
| `rating` | Возрастной рейтинг ESRB (E, E10+, T, M, unknown и др.) |

## 🔍 Задачи исследования

1. **Предобработка данных**
   - Приведение названий столбцов и текстовых данных к единому формату
   - Обработка пропусков и дубликатов
   - Преобразование типов данных
   - Создание столбца с общими продажами

2. **Исследовательский анализ данных**
   - Анализ количества игр по годам
   - Определение жизненного цикла платформ
   - Выбор актуального периода для прогнозирования
   - Выявление перспективных платформ
   - Анализ влияния оценок на продажи
   - Исследование распределения игр по жанрам

3. **Портрет пользователя по регионам**
   - Северная Америка (NA)
   - Европа (EU)
   - Япония (JP)
   - Другие страны (Other)

4. **Проверка статистических гипотез**
   - **H1**: Средние пользовательские рейтинги платформ Xbox One и PC одинаковые
   - **H2**: Средние пользовательские рейтинги жанров Action и Sports разные

5. **Формирование выводов и рекомендаций**

**Вывод:** Shooter — самый прибыльный жанр в расчёте на одну игру, Action — самый популярный по количеству игр.

## 💡 Рекомендации для бизнеса

### 1. Фокус на платформы
- **PS4** и **Xbox One** — наиболее перспективные платформы для рекламной кампании на 2017 год
- Имеют высокий потенциал роста и устойчивый тренд продаж

### 2. Качество важнее количества
- Оценки критиков имеют умеренную положительную корреляцию с продажами (0.41)
- Рекомендуется инвестировать в получение высоких оценок от профессиональных критиков
- Оценки пользователей не являются значимым фактором успеха

### 3. Жанровая стратегия
- **Shooter** — самый прибыльный жанр на одну игру
- **Action** — самый популярный жанр по количеству игр
- **Sports** — стабильно высокие продажи
- Рекомендуется фокусироваться на этих жанрах для максимизации прибыли

### 4. Учёт региональных особенностей
| Регион | Особенности |
|--------|-------------|
| Япония | Предпочтение портативным платформам (3DS, PSV) и жанру Role-Playing |
| Европа/США | Лидируют стационарные консоли и жанры Action/Shooter |
| Все регионы | Игры с рейтингом M (для взрослых) имеют наибольший коммерческий успех |

### 5. Ключевые метрики успеха
- **Высокая оценка критиков** → выше продажи
- **Жанр Shooter/Action/Sports** → выше продажи
- **Платформа PS4/Xbox One** → выше продажи
- **Рейтинг M** → выше продажи
