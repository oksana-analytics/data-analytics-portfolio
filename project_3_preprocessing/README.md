## 📌 Описание проекта

Анализ данных сервиса аренды самокатов **GoFast**. Исследование поведения пользователей с подпиской **Ultra** и без подписки, проверка статистических гипотез для оптимизации бизнес-стратегии.

### 🎯 Цель исследования
Определить, являются ли пользователи с подпиской более выгодными для компании, и дать рекомендации по увеличению их количества.

---

## 📊 Данные

Проект использует три набора данных:

| Файл | Описание |
|------|----------|
| `users_go.csv` | Информация о пользователях (возраст, город, тип подписки) |
| `rides_go.csv` | Данные о поездках (расстояние, длительность, дата) |
| `subscriptions_go.csv` | Тарифы (цены для free и ultra) |

---

## 🔍 Задачи исследования

1. **Исследовательский анализ данных**
   - Распределение пользователей по городам
   - Соотношение подписчиков и пользователей без подписки
   - Анализ возраста, расстояния и продолжительности поездок

2. **Проверка статистических гипотез**
   - **H1**: Пользователи с подпиской тратят больше времени на поездки
   - **H2**: Среднее расстояние поездки подписчиков не превышает 3130 метров
   - **H3**: Помесячная выручка подписчиков выше, чем у пользователей без подписки

3. **Расчёт выручки**
   - Подсчёт помесячной выручки для каждого пользователя
   - Анализ общей выручки сервиса за 2021 год

## 📈 Ключевые выводы

### 1. Исследовательский анализ
| Показатель | Результат |
|------------|-----------|
| Город с максимумом пользователей | Пятигорск (219) |
| Город с минимумом пользователей | Москва (168) |
| Соотношение подписок | 54% free / 46% ultra |
| Средний возраст пользователя | 25 лет |
| Средняя дистанция поездки | ~3 км |
| Средняя продолжительность | 17 минут |

### 2. Проверка гипотез

**Гипотеза 1:** *Пользователи с подпиской тратят больше времени на поездки*
- **Результат**: ✅ **Отвергаем H0**
- **p-value**: 2.5e-34 (<< 0.05)
- **Вывод**: Время поездок у подписчиков статистически значимо больше

**Гипотеза 2:** *Среднее расстояние подписчиков не превышает 3130 м*
- **Результат**: ❌ **Не отвергаем H0**
- **p-value**: 0.912 (> 0.05)
- **Вывод**: Недостаточно оснований утверждать, что расстояние превышает оптимальное

**Гипотеза 3:** *Выручка подписчиков выше, чем у пользователей без подписки*
- **Результат**: ✅ **Отвергаем H0**
- **p-value**: 1.89e-31 (<< 0.05)
- **Вывод**: Подписчики приносят больше выручки в расчёте на пользователя

### 3. Выручка
- **Общая выручка за 2021 год**: **3 878 641 руб.**

2. **Адаптировать маркетинговые кампании**

3. **Оптимизировать износ самокатов**
   - Среднее расстояние подписчиков близко к оптимальному (3130 м)
   - Специальные условия для поездок в пределах оптимальной дистанции
